# Prompts Used for ChatGPT

To generate the Python script (`fim_fim_predict.py`), the following prompt was provided to ChatGPT (GPT-4):

"Write a Python script to predict discharge Functional Independence Measure (FIM) scores using Linear Regression. The input variables are age, onset-to-admission days, admission motor FIM, and admission cognitive FIM. Outcomes are discharge motor FIM, cognitive FIM, and total FIM. Include data preprocessing, 5-fold cross-validation, and performance evaluation metrics (MAE, RMSE, R-squared, calibration intercept and slope). Output the results to an Excel file. No patient data will be provided."
